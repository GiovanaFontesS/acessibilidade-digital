# HTML Semântico

---

O **HTML semântico** consiste na utilização de elementos que descrevem o significado e a função do conteúdo de uma página, em vez de apenas definir sua aparência. Dessa forma, cada parte do documento possui uma estrutura clara e organizada, facilitando a interpretação tanto pelos navegadores quanto por mecanismos de busca e tecnologias assistivas, como os leitores de tela.

Ao utilizar tags semânticas, como `<header>`, `<main>`, `<nav>`, `<section>`, `<article>` e `<footer>`, é possível criar páginas mais acessíveis, organizadas e fáceis de navegar. Essa estrutura permite que usuários que utilizam tecnologias assistivas compreendam melhor a hierarquia das informações e encontrem o conteúdo desejado com maior facilidade.

Além de contribuir para a acessibilidade digital, o HTML semântico melhora a manutenção do código, favorece a indexação em mecanismos de busca (SEO) e promove boas práticas no desenvolvimento web. Por isso, seu uso é recomendado pelo **W3C** e está diretamente relacionado às diretrizes de acessibilidade da **WCAG**, sendo um dos principais recursos para a criação de aplicações inclusivas.

Nos próximos tópicos serão apresentadas as principais tags semânticas do HTML e sua importância para o desenvolvimento de páginas acessíveis.
